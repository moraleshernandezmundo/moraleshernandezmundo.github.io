Programacion

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea. Pero no solo la acción de escribir un código para que la computadora o el software lo ejecute.

Diseño

En el ciclo de vida de un programa, una vez que los requisitos de un programa han sido establecidos en la fase de análisis, ya se puede iniciar la fase de diseño. En esta etapa se tiene que encontrar una solución informática al problema planteado. Dicha solución determinará cómo se va a resolver el problema.

¿Qué es Programacion?

Programación es el proceso de crear, diseñar, escribir, probar y depurar código para que una computadora u otro dispositivo se comporte de cierta forma y obedezca órdenes. Para ello, se usa un lenguaje de programación que es una serie de normas, instrucciones y expresiones semejantes a una lengua natural acotada. La programación también puede referirse a la acción de ordenar, estructurar o componer una serie de acciones cronológicas para cumplir un objetivo en diferentes ámbitos.

¿Qué aprenderé en este curso?

Todo lo que utilizamos, como ordenadores portátiles, máquinas expendedoras y automóviles, funciona con código. Pero ¿sabes lo que es? En estos vídeos, te familiarizarás con el concepto de "código", descubrirás porqué hay tantos lenguajes de programación y cómo se compaginan para hacer tareas específicas. Sobre todo, entenderás porqué tener conocimientos básicos de programación puede serte útil.

¿Es facil aprender a programar?

Esta es una de las preguntas mas frecuentada, ya que las programacion es algo conplicado de enteder, por la logica que se usa, pero todo consiste en el alumno, porque cada quien tiene una manera de progrnmar y dar logicas a sus programas, pero poniendo mucha atencio a los cursos, y estudiando de maneras externas, podras facilitar tu manera de programar astas familiarizando con la programacion.

¿Qué tan pronto aprendere a programar?

Si estás comenzando desde cero, no estás yendo a una universidad relacionada y nunca has tenido experiencia con la tecnología, es un largo camino. Tu primer curso, tu primer libro, sólo te mostrará los pasos iniciales. La práctica es que te dará fluidez, como dijimos. Un año después de tu inicio, es un buen momento para decir que puede que hayas cumplido las condiciones para empezar a trabajar con tu primer lenguaje. Por supuesto, esto variará mucho de una persona a otra.
